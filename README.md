<b>2、Android系统的Tiktok是会检测SIM卡信息</b>

如果是国内的手机卡会直接拒绝网络访问的,会提示网络连接错误。

##### 1、什么是Shizuku？

这里科普一下什么是Shizuku？


![](https://files.mdnice.com/user/4562/be30f416-71d9-42b1-9813-c10acbfdf92f.png)


大名鼎鼎的开源的 ADB 权限管理工具，旨在为安卓用户提供一种无需获取完整 ROOT 权限即可管理高级系统权限的解决方案。

Shizuku主要解决这了2个重要功能：

1、为广大手机发烧友提供一个方便地使用系统API的方式（系统提权）。

2、为部分只需要adb权限的应用开发提供便利。

Shizuku在这里的作用就是提供特权服务。

# 4、Nrfr相关问答


1、Nrfr、Shizuku是否安全？

这两款软件都开源好多年了，基本不存在任何安全问题，经得起时间的考验。

Shizuku开源地址：

https://github.com/RikkaApps/Shizuku


![](https://files.mdnice.com/user/4562/720bf688-c79f-4f69-90f1-55b73632c21a.png)

Github上标星16.2k，足以可见其受欢迎程度。

Nrfr开源地址：

https://github.com/Ackites/Nrfr

目前在Github标星4.5K，知名度肯定小于Shizuku。

2、点击版本号5次还是无法出现开发者选项？

有的Android系统可能需要点击7次。

3、出来Tiktok会检测Sim，还有哪些App检测？

Samsung Health，三星推出的健康管理App，支持Galaxy Watch、Galaxy Fit3等可穿戴设备，提供睡眠监测、运动追踪、心率检测等功能。

使用Samsung Health首次会进行 SIM 卡检测。

4、Nrfr是能修改所有安卓设备的SIM信息？

部分安卓手机可能不支持修改国家码！（这个非常少）

5、如何使用第三方App检查是否修改成功？

可使用“设备信息”这款App检查查看，如图所示：

![](https://files.mdnice.com/user/4562/3ace1431-13a6-46d9-8b29-3664d5c63487.png)

下载地址：

https://pan.quark.cn/s/aa006afb76bb

所在文件路径「Android软件合集」>「Android实用工具合集」>「5款查看手机硬件信息+检测」。

![](https://files.mdnice.com/user/4562/a4b2da7e-517d-405e-b9e2-f70702bcbe9c.png)


6、修改完成之后，是否需要重启？

修改完成后无需重启设备，设置会立即生效并永久保持。还原正常SIM必须使用Nrfr内的还原功能。

7、Nrfr的实现原理？

Nrfr 仅改变系统对 SIM 卡信息的读取方式，通过调用 Android 系统级 API（CarrierConfigLoader）修改系统内的运营商配置参数，而不是直接修改 SIM 卡。

完全在系统层面工作，不会对 SIM 卡本身进行任何修改或造成损坏。

如需还原设置，请使用应用内的还原功能。

8、使用Nrfr修改无效？

检查Shizuku是否正常启动？

检查Shizuku是否授权给Nrfr？

检查Nrfr是否正确修改对应的Sim卡（如果有2个卡）。

9、修改完成之后，打开Tiktok还是显示网络错误？

请检测TiZi的节点，更换一下节点试试，注意不要切换到香港地区，香港地区也无法直接下载运行Tiktok。
